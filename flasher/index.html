Quick commands
Select profile, tweak if needed, then Apply + Save.
Morse blinker
Run radio diagnostics to debug RX/TX issues. Results appear in the console below.
rxdiag SX1262 RX state, chip mode, IRQ flags, BUSY pin
nfloor Noise floor RSSI — test antenna connection
regdump Read SX1262 freq/sync/pkt registers
irqmon 20s Monitor IRQ flags for 20s — detect preambles
pktdump on Hex-dump every received packet
pktdump off Stop verbose packet dumping
rxraw Force-read SX1262 FIFO contents
txraw Transmit raw hex bytes (no header/CSMA)
reinit Re-run full radio initialization sequence with discovery phases
Quick reference
RX Debug Sequence: rxdiag → nfloor → regdump → irqmon 30
Antenna check: nfloor should show -100 to -115 dBm (< -125 = broken)
Sync word: 0x12 / 0x1424 = Reticulum  |  0x34 / 0x3444 = LoRaWAN

🚀 Flash firmware
The device will reboot into DFU mode. On the first flash, you may need to click Write to device drive once; later flashes in this tab can continue automatically.

Put device into DFU mode manually:
  1. Double-tap the reset button on the WisBlock.
  2. A USB drive named RAK4631 (or similar) will appear on your computer.
  3. Click "Write to device drive" below and select that drive.
On first use, a folder picker will open — select the RAK4631 USB drive
